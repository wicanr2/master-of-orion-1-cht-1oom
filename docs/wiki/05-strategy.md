# 05 策略指南

《銀河霸主》的勝負,往往在開局幾十回合就埋下伏筆。這一篇從新手入門開始,談種族選擇、早期擴張、科技路線、外交與戰爭,以及怎麼用銀河議會收尾。原則不複雜:**先穩穩擴張、把科技與經濟養起來,再決定要靠票數還是艦隊終結對手。**

## 新手三步驟

1. **選對種族**:新手從容錯高的種族入門——**賽隆**(科技碾壓)、**克拉肯**(早期經濟強)、**人類**(外交穩健)。
2. **前期猛擴張**:開局交替建造**偵察艦**(點亮星圖)與**殖民船**(佔行星),搶下身邊每一顆能住的行星。
3. **科技別斷**:每顆殖民地盡早蓋研究設施,規劃一條科技路線持續往上推,別讓研究停擺。

## 種族選擇思路

| 你想要的玩法 | 推薦種族 |
|---|---|
| 科技滾雪球、和平發展 | 賽隆 Psilon |
| 爆產能、人海推進 | 克拉肯 Klackon、梅克拉 Meklar |
| 外交/貿易/議會 | 人類 Human |
| 快速鋪盤擴張 | 薩克拉 Sakkra |
| 獨特機制、佔惡劣星 | 矽基 Silicoid |
| 正面開砲壓制 | 莫善 Mrrshan |
| 諜報陰謀 | 達洛克 Darlok |
| 地面攻城 | 布拉西 Bulrathi |

種族特性詳見 [02 種族](02-races.md)。

## 早期:探索與擴張 (約前 50 回合)

- **交替造偵察艦與殖民船**:偵察艦點亮星圖、找出可住與富礦行星;殖民船立刻去佔。
- **研究優先推進 + 行星學**:
  - 看到「**可住但偏遠**」的行星 → 衝**推進**(增程燃料)把它納入射程。
  - 身邊多是「**貧瘠/凍原等惡劣**」行星 → 衝**行星學**(環境控制)解鎖殖民資格。
- **約一半的銀河行星是惡劣環境**,而這些惡劣星往往是**富礦 (Rich)**——後期造艦的本錢。早期研 2~3 項行星學去吃下它們,長期回報極高。
- **搶位優先於完美**:先佔住地點,再慢慢養;晚一步可能就被 AI 搶走或夾在中間。

## 中期:開發與軍備

- **減廢是分水嶺**:建造線的**減廢工廠**讓你不必把產能浪費在清汙染上,中期產能直接解放(矽基天生免疫,跳過此線)。
- **建研究設施 + 規劃科技路線**:用科技畫面選一條連續路徑,持續研究,別讓任何一局停在原地。
- **看牌打**:每局只亮出部分科技,缺的關鍵科技就靠**外交交換**或**間諜偷取**補。
- **開始建軍**:中期要有能自保的艦隊與行星防禦(飛彈基地 + 行星護盾),否則容易被 AI 趁虛而入。

## 外交:跟 AI 打交道

- **AI 很雙標**:即使簽了**互不侵犯協定 (Non-Aggression Pact)**,AI 的偵察艦飛過你殖民地後,常常跟著就是運兵船入侵;而你卻無法透過外交介面對它宣戰。**一旦你還手,它立刻翻臉開戰。**
- **善用貿易**:人類靠貿易協定回報 +25% 滾經濟;多開貿易路線也能改善關係。
- **離間與賄賂**:可以付錢請某個帝國去打另一個,或挑撥兩個 AI 互鬥,自己坐收漁利。
- **盯緊鄰居實力**:外交關係會隨勢力消長浮動;別讓任何一個 AI 無人牽制地坐大。

## 戰爭要點

- **早期飛彈/魚雷,後期光束**:早中期飛彈與魚雷 CP 值高;後期靠**戰鬥電腦 + 高能聚焦**的光束流派輸出最穩。
- **電腦配光束**:用光束務必堆戰鬥電腦,否則命中慘澹。
- **行星防禦鐵桶陣**:飛彈基地 + 行星護盾 + 兩艘斥力光束艦放行星兩側,可封鎖敵方投彈路徑,殖民地極難被攻陷。
- **地面登陸**:轟炸削減守軍後派運兵船登陸搶星;布拉西地面戰 +25,一次能換掉數倍敵軍。
- **別過度延伸**:戰線拉太長、補給跟不上,反而被反推。集中兵力打關鍵星系。

## 收尾:銀河議會還是艦隊

當某帝國夠大,**銀河議會 (Galactic Council)** 會召開投票,票數依各帝國控制的人口規模分配:

- **取得全銀河三分之二票數**即當選銀河霸主,遊戲結束。
- 想靠**議會勝**:長期經營人口與外交,廣結貿易與盟友(人類最擅長)。
- 若你**被選為非霸主**(別人當選),可以**拒絕承認**、繼續武力周旋,但代價是與當選者及其支持者全面開戰。
- 想靠**軍事勝**:就把對手一個一個抹掉,直到獨佔銀河。

> 實戰常見是兩者交織:先用科技與外交壯大、累積人口與盟友,再臨門決定——票數夠就投票收尾,不夠就艦隊收尾。

## 常見新手錯誤

- **科技停擺**:只顧造船不蓋實驗室,中期被科技差距碾壓。
- **不吃惡劣行星**:放掉富礦惡劣星,等於放掉後期造艦本錢。
- **小型艦撐到後期**:後期小型艦一發必死,該換大型/巨型。
- **光束不配電腦**:命中低到打不痛人。
- **輕信 AI 協定**:互不侵犯不等於安全,隨時備戰。
