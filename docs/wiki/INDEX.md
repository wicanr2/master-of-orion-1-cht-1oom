# 銀河霸主小百科 (Master of Orion 1 Wiki)

## 關於本百科

- **正確性優先**:遊戲數據(加成幅度、傷害值、護盾等級等)以多個攻略站與社群資料交叉比對為準;少數無法確認的細節標註「待考」。
- **版權**:所有內容為自行改寫摘要,非原文照搬;各篇結尾附參考來源 URL。
- **譯名一致**:種族、科技、機制譯名與本專案 `CONTEXT.md` 及中文化字串對齊。
- **版本**:以原版 1.3 (DOS) 為基準。本專案使用的 `1oom` 引擎重現原版玩法,數值與原版一致。

> 名稱小註:MOO1 常見譯名有「銀河霸主」「奧瑞恩的霸主」。本專案統一採「銀河霸主」。
